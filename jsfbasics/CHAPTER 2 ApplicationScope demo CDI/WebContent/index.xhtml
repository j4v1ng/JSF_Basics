ApplicationScoped demo CDI(Component Dependency Injection)

EXPLANATION:
It you submit a value, it will be submited and displayed at the destination page.
Since the backing bean is @ApplicationScoped, the value will remain saved while the application
is running on the server. The values of an ApplicationScoped backingbean, are attached to the application,
and not to a concrete user.
