CHAPTER 9 - Composite components

Composite components will allow you have reusable components that can be packed and reused in other applications.
The steps to create composite components are:
1. Create a composite component under the resources(You can create sub package called ccs) package and add the name space xmlns:composite="http://java.sun.com/jsf/composite
2.Every composite component has 2 parts: interface: Atributes that the component can use will be specified here
implementation: The markup implementation of the component
