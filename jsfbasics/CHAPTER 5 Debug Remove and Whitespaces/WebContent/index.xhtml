Debug Remove and Whitespaces

Debug using ui:debug tag

If you place the tag ui:debug somewhere in a page, you will see the
component tree of this page.A recomentadion is to put the debug tag on the template


When the application is running in the browser, press CTRL+SHIFT+D to open the component tree Note: When the application is finished you should remove all debug tags.

Using ui:remove tag as comments

HTML comments on JSF, have no effect on components(only sometimes), the way things should be commented is using the remove tag

White spaces

To place white spaces you should use empty value expresions

#{' '}
