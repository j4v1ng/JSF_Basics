Bindings and Id's'

Bindings

Often we use the attribute value of the component tags, to assign a given input to a variable in a backing bean
But sometimes we need to save not just the inputed value, but the hold component object
In that case we use binding(See the example)




ID's

Every JSF tag has an id. Id's are useful for:

  • Access JSF Components from other JSF tags
  • Obtain Component references in Java code
  • Access HTML elements with scripts
